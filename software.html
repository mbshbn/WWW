Fourier Transform on SO(3)

This open-source C++ software package provides various tools for harmonic analysis on the special orthogonal group and it is available at

https://fdcl-gwu.github.io/FFTSO3

  • What does it do ?
    • Complex/Real Noncommutative Harmonic Analysis on SO(3)
    • Complex/Real Spherical Harmonics on the Unit-Sphere
    • Fast Fourier Transform on SO(3)
    • Compute Wigner D-matrix
    • Compute Clebsch-Gordon coefficients
    • Parallel computing with OpenMP
